Интерактивная игра "Путешествие в азбуке финансов" на русский манер.

В рамках нее ведущий нажимает на кнопку и появляется один из вопросов
- тест по финграмотности с тремя вариантами ответа
- творческий вопрос (нужно действие сделать либо перечислить что-то)
- алфавитный вопрос (вопрос на размышление о каком-то финансовом термине/аспекте)

Игра несет познавательный аспект, когда ведущий спросил - получил ответ и объяснил это аудитории
